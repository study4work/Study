ИСПОЛЬЗОВАТЬ BYTE STREAMS ДЛЯ:    
✅ Бинарные файлы:  
   • Изображения (jpg, png, gif)   
   • Аудио (mp3, wav)  
   • Видео (mp4, avi)  
   • Архивы (zip, rar)   
   • Сериализованные объекты  
   • PDF, DOC, XLS       
                                                     
✅ Сетевые данные (когда работаем с сырыми байтами)
                                                     
✅ Когда кодировка не важна                         
                                                     
❌ НЕ ДЛЯ ТЕКСТА!   
   • Текстовые файлы нужно читать через Reader  
   • Иначе будут проблемы с кодировкой
