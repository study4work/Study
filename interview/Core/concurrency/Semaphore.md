## Semaphore
—  счётчик разрешений. Это не лок, а счётчик разрешений. Представьте парковку на 5 мест — семафор с permits=5.

НЕ реализует Lock (нет reentrancy!)  
Задача: ограничить число потоков, одновременно работающих с ресурсом  
Состояние AQS: int-счётчик разрешений

```java
Semaphore semaphore = new Semaphore(3);  // 3 разрешения

public void accessResource() throws InterruptedException {
    semaphore.acquire();   // запросить разрешение (если 0 — ждать)
    try {
        // работаем с ресурсом (не более 3 потоков одновременно)
    } finally {
        semaphore.release();   // вернуть разрешение
    }
}
```

Главное отличие от лока:  
Лок: один поток владеет ресурсом  
Семафор: N потоков могут владеть ресурсом одновременно
